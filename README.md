# Finance Dashboard

A responsive finance dashboard built with React, Tailwind CSS, Recharts, and browser `localStorage`.

## 🌐 Live Demo

[![Live Demo]([https://img.shields.io/badge/Live%20Demo-Click%20Here-blue?style=for-the-badge)](https://zorvyn-project-szqo-ox7rlhxmm-ankit8jsts-projects.vercel.app/](https://zorvyn-project-szqo.vercel.app/))

## Run the project

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build for production

```bash
npm run build
npm run preview
```

## Features

- Summary cards for balance, income, and expenses
- Responsive line and pie charts
- Transaction search, type filter, and sorting
- Admin-only add transaction form
- Role switcher between viewer and admin
- Dark mode toggle
- Persistent data with `localStorage`
